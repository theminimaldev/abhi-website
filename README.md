# abhishek jangra — portfolio

Personal portfolio website built with HTML and CSS.

## Stack

- Plain HTML + CSS (no frameworks)
- Inter font via Google Fonts
- Light/dark theme with time-based auto-detection

## Development

Just open `index.html` in a browser. No build step needed.

## Deployment

Hosted via GitHub Pages at [theminimaldev.com](https://theminimaldev.com).
